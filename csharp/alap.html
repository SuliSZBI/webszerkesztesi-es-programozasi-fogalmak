c#: Alap C# forráskód

using System;

namespace Pelda
{
    internal class Program
    {
        static void Main(string[] args) 
        {
            Console.WriteLine("Hello World!");

            Console.ReadKey(true);
        }
    }
}

using System;

A program futtatásához szükséges névterek (namespace) beimportálása.

namespace Pelda

A projekt készítéseko adott névvel létrehozott névtér, amely majd tartalmazni fogja az osztályainkat (class), interfészeinket (interface) stb.

internal class Program

A program indítópontját tartalamzó osztály. Általában Program a neve.

static void Main(string[] args)

A program indítópontját reprezentáló metódus.

Console.WriteLine("Hello World!");

Kiíratás a konzolra (standard output).

Console.ReadKey(true);

Konzol bezáródásának a megakadályozása.
