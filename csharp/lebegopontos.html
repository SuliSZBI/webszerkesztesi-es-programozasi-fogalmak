c#: Lebegőpontos számok

using System;

namespace Pelda
{
    internal class Program
    {
        static void Main(string[] args) 
        {
                float adat_01 = 169.15F;
		Console.WriteLine($"Single (±1.5 x 10-45 - ±3.4 x 1038) érték: {adat_01}");
		double adat_02 = -16.48D;
		Console.WriteLine($"Double (±5.0 × 10-324  - ±1.7 × 10308) érték: {adat_02}");
		decimal adat_03 = 15896.8965M;
		Console.WriteLine($"Decimal (±1.0 x 10-28  - ±7.9228 x 1028) érték: {adat_03}");

            	Console.ReadKey(true);
        }
    }
}

float

4 byte-on reprezentált lebegőpontos értéktípus (floating type).

Értékkészlet: ±1.5 x 10-45 - ±3.4 x 1038

.NET megfelelője: System.Single

double

8 byte-on reprezentált lebegőpontos értéktípus (floating type).

Értékkészlet: ±5.0 × 10-324 - ±1.7 × 10308

.NET megfelelője: System.Double

decimal

16 byte-on reprezentált lebegőpontos értéktípus (floating type).

Értékkészlet: ±1.0 x 10-28 - ±7.9228 x 1028

.NET megfelelője: System.Decimal
